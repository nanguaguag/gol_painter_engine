## feature

- 2197\*2187的网格地图，够大
- 采用vim的模式（瞎搞的）

## 缺点

- 占用内存很大，动辄几百MB
- 地图不是无限大的（能力不足）

## 快捷键

- [SPACE] 开始/暂停
- [ H ] 帮助信息
- [ G ] 显示网格
- [ESC] 进入移动模式
- [ I ] 编辑模式
- [ V ] 选择模式
- [ - ] 减速
- [ = ] 加速
- [BACKSPACE] 清空所选区域
- [LEFT] 放大
- [RIGHT] 缩小
